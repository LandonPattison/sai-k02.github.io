MICROSOFT/UTD Decentralized Verifiable Credentials

January 30 2022

This project was a part of Microsoft's Decentralized Identity Hackathon. The goal of the hack is to "build any type of app (mobile, web, desktop) using your preferred programming language that uses the Verifiable Credential REST API to issue and verify VCs." This is a simple .NET web app that simply showcases just how in seconds you can be using Verifiable Credentials in your own use case! Received "Azure Identity and Access Management 2022" Credential for placing top ten. Created with Azure Active Directory Verifiable Credentials, Blob Storage, Key Vault, HTML/CSS, Basic .NET Project., REST API, VS Code, Github/Github Actions.
